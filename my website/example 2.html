A bit about the internet

URL

URL stands for *Uniform Resource Locater*It contains several different parts , including:

  • the protocol: e.g. http, https, ftp
  • the host: e.g. link to facebook, link to bbc
  • the path: e.g. /news/local, /john.smith.95

DNS

DNS stands for Domain Name System. You buy a domain name from a registrar (e.g. link to 123-reg , link to GoDaddy) they arrange for your server's IP address to added next to the domain name on a DNS server, so that people can find your site. this process can take a couple of days to spread out over the entire internet, as DNS servers take time to update one another using a 'gossiping' mechanism.
